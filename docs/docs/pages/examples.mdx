# Examples [Demo examples for Dacc-js functions]

Interactive examples demonstrating various Dacc-js functions and use cases. These examples provide hands-on experience with [code](https://github.com/thefactlab-org/dacc-js/tree/main/examples) that you can test and modify.

## Client-side

Client-side examples showcase how to use Dacc-js in **frontend** applications using React and Vite for real-time, interactive blockchain operations.

:::note
Recommendation: This test is on **optimismSepolia** Network
:::
1. Click on `Preview` or `Both` on StackBlitz tab.
2. You should wait 1-5 minutes to load.
